Question-1: What are the different ways to select an element in the DOM?

Different Ways to Select Elements in the DOM .
In JavaScript, there are multiple ways to select elements from the Document Object Model (DOM). Each method has its own advantages and use cases. Among them, my personal favorite and the most convenient one is getElementById(). However, there are several other methods as well. The getElementById(id) method selects a single element based on its unique id. This is one of the most efficient ways to access an element directly. The getElementsByClassName(className) method selects multiple elements that share the same class. It is useful when applying changes to a group of elements with a common class. The getElementsByTagName(tagName) method selects all elements of a specific tag type, such as div, p, or span. This is helpful when working with multiple elements of the same type. The querySelector(selector) method selects the first element that matches a given CSS selector. It allows for flexible selection using class names, IDs, or element types. Question-3: What is event delegation in the DOM?

Event delegation is a DOM technique that helps in dynamically controlling a website. It works by using event listeners, such as the event.target property, which is used to identify the element from which the event originated. Through this, you can easily manage events for multiple child elements by attaching a single event listener to a parent element.

Question-4: What is event bubbling in the DOM?

If you click on a button, and the button is inside a parent element, the click event will first occur on the button, and then it will bubble up to the parent element, its parent, and even the entire document. You can add an event listener to the parent element using event bubbling.
